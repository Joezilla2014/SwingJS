SwingJS is a tool for rapidly and easily converting Java Swing applets to JavaScript.

Why bother converting Java Applets to JavaScript?

Java Applets for web applications have become outdated and clumsy, requiring the user-install of Java, updated security certificates, and poor browser-side support. However, they provide a useful tool, allowing for complicated simulations and tasks that would usually be impossible on the web. By converting these Java Applets to JavaScript, we maintain the functionality of the applets, therby preserving their usefulness, without any of the clunkiness.

This website was created to demonstrate and archive the applets that have been successfully converted to JavaScript.
